## Для запуска проекта:

Фронтенд:
npm run start — запускает сервер в dev-режиме
npm run build — собирает приложение

Бэкенд:
npm run start — запускает сервер
npm run dev — запускает сервер с hot-reload


## Технологии проекта

Frontend: React

Backend: NodeJS, Postman, Express, MongoDB
